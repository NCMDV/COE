STEPS TO CONVERT FORM TO JSON DATA:

1. On your form page, click on the 3 dots and select "Script Editor"
2. Authorize the script if prompted
3. Paste the code inside "script_v2" to "Code.gs"
4. Replace <URL> with your form edit url
5. Save the script, on the dropdown near the Debug button, select "main"
6. Click run
7. On the Execution log, copy the value to the right of "Info"
8. Paste on a Notepad, Click File then "Save as"
9. Set File Name to "your_desired_filename.json"
10. Change the "Save as type" to "All Files"
11. Click Save
